## 🎨 Section 1: Dynamic Material 3 UI/UX Design Spec

The visual design system of the **MARCO Personal Assistant** utilizes an elegant, high-contrast, mobile-first, adaptive interface meticulously crafted with Jetpack Compose.

### 🌓 1. The Divine Cosmic Slate Theme
- **Background Palette**: Dynamic dark slate color codes (`Color(0xFF0F111A)`) that minimize eye fatigue.
- **Surface Elevation**: Rounded cards styled in high-spec elevated slate outlines with smooth drop-shading.
- **Core Accent Color**: Radiant Electric Violet (`Color(0xFF8B5CF6)`) for action triggers, message badges, and focus frames.
- **Dynamic Content Contrast**: Crisp off-white message bubbles (`Color(0xFFE2E8F0)`) for maximum readable contrast, keeping communication beautifully fluid.

### 📱 2. Core Layout & Interactive Modules
- **Modular Title Header (`TopAppBar`)**: Features a clean minimalist title of MARCO in dynamic contemporary typography, aligned next to safe status LEDs that trace database/network readiness.
- **Adaptive Message Flow**: Dynamic chat screen fueled by Room local database caching, scrolling with stateful vertical physics.
- **Interactive Suggestion Shelf**: Custom horizontal scrollbar containing elegant lightbulb chips that automatically paste highly advanced domain-expert queries with a pleasant tactile feedback (Material Ripple).
- **Premium Control Field**: Features rounded filled text inputs with seamless edge-to-edge padding, coupled with a bold circular send fab utilizing vector-designed graphics assets.
