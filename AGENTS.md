## 🎨 1. Structural Visual Theme (The Cosmic Slate Specs)
- **Primary Canvas Background**: Dark Cosmic Slate (`Color(0xFF0F111A)`). Always avoid blinding lighter shades.
- **Accents & Action Rings**: Radiant Electric Violet (`Color(0xFF8B5CF6)`) and soft Shariah Cyan accents (`Color(0xFF009688)`).
- **Text Bubbles**: Crisp high-contrast off-white styling (`Color(0xFFE2E8F0)`) inside elevated card enclosures (Tonal Elevation 6dp).

## 🎙️ 2. Hands-Free Voice Assistant & Wake-Word
- **Voice Wake-Word detection**: Listens for `"Hey Marco"`, `"Marco"`, `"মারকো"`, `"হাই মারকো"`, or `"ওহে মারকো"` to activate the main conversational state.
- **Continuous Conversation Pipeline**:
  - Automatically restarts recognition listener once voice speaking (Text-To-Speech) finishes.
  - Seamlessly handles English (UK/US) and Bengali (বাংলা) voices with natural accents.
- **Voice Dashboard Command Keywords**:
  - `clear chat` / `পরিষ্কার` -> Resets message database histories.
  - `tips` / `টিপস` -> Opens the Lightbulb Tips sheet.
  - `download` / `ডাউনলোড` -> Spawns PC installer dialogue matrix.
  - `exit assistant` / `বন্ধ` -> Closes the voice dashboard.

Orchestrates autonomous simulated sub-agents across a 5x5 coordinate board:

4. **Domestic Bot (Cleaner 🧹 - Orange)**: Sweeps cache systems and purges obsolete background memory tracks.

- **Pricing Web Builder (`WEBSITE`)**: Carol codes the components (`progress 0% -> 50%`), transfers code blocks, and Sophia audits financial compliance (`progress 50% -> 100%`).
- **Outbound Email/SMTP (`PAYMENT`)**: Sophia verifies ledger transaction deposits, passes context, and Brian broadcasts automated receipts.
- **Memory Sweeper (`CLEANER`)**: Domestic Bot wipes background databases and resets storage layouts.
